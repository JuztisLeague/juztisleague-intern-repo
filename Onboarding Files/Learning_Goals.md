# Learning Goals during Focus Bear Intership

## Reflection

During this internship, I hope to learn both technical and professional skills that will help me grow as a developer.

Specifically, I want to learn:

- Mobile application development, particularly using React Native and Expo, since this is the technology used in the Focus Bear mobile app.
- Real-world software development workflows, including Git collaboration, pull requests, code reviews, and team communication.
- How to contribute to an existing codebase, which is different from building personal projects from scratch.
- How productivity tools like Focus Bear can help users, especially those who struggle with focus or executive functioning.

In addition to technical knowledge, I also want to improve my ability to communicate updates clearly to teammates. For example, I want to be able to explain what changes I made in my code, why I made them, and how they affect the application so that my peers can quickly understand my work.

Ultimately, I hope to give my full effort during the internship so that I can gain enough experience and confidence to pursue a software development job after completing the program.

The main skills I want to develop during this internship include:

- Code structure and organization
I want to improve how I structure my code so that it is easier for others to read and maintain. For example, I want to learn how to properly organize components, hooks, and files in a React Native project.

- Explaining technical work clearly
I want to become more confident when explaining the improvements or changes I make in my code. For example, during code reviews or pull requests, I want to be able to clearly describe what my changes do and why they were necessary.

- Problem-solving and debugging
I want to improve my ability to debug issues in applications. For example, if a component does not render correctly or an API request fails, I want to learn systematic ways to identify and fix the problem.

- Consistency and focus while coding
I want to develop better habits when working on coding tasks so that I can stay productive even when tasks become difficult.


Even though I am still learning, I believe I can contribute to the team in several ways:

- Sharing ideas for improving the application
For example, I can suggest improvements for the user interface or user experience of the mobile or web application.

- Testing and troubleshooting features
I can help identify bugs or usability issues by testing features and reporting any problems I encounter.

- Supporting team discussions
If I notice something that could improve the application or development process, I can share those ideas with the team.

- Completing assigned development tasks
As I improve my technical skills, I hope to contribute by implementing smaller features or fixing bugs in the codebase.


One of my current challenges is that I am not fully confident in my coding skills yet. While I understand programming concepts and have experience with languages like Java, C, and JavaScript, I feel that I still need more practice working on real-world projects.

Another challenge is that I sometimes lose motivation or become bored when working on difficult coding tasks. However, I am trying to overcome this by reminding myself why I started learning programming in the first place. I see this internship as an opportunity to reignite my passion for coding and improve my discipline when working on projects.

By consistently practicing and contributing during this internship, I hope to build more confidence in my abilities as a developer.


For me, success in this internship means:

- Learning new technical and professional skills

- Becoming more confident in my coding abilities

- Contributing meaningful work to the team

- Improving my communication with teammates

A major personal goal is to use what I learn during this internship to help me secure a software development job in the future. If I can continue working with Focus Bear after completing my internship hours, that would be a great achievement for me.

Regardless of the outcome, I am grateful for the opportunity to be part of the Focus Bear team, and I want to use this experience to improve myself both professionally and personally.
