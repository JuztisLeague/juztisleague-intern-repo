# Setting up a React Native Development Environment (Expo & Metro Server)

So the role of Metro in React Native development is that it is a JavaScript bundler. Therefore, its job is to bundle hundreds of separated files into one JavaScript file. Also, it is for transformation since it has the ability to transform JavaScript/TypeScript into a format for the mobile phone JavaScript Engine.

Expo simplifies React Native Development through Zero-Config Native Code, which you don't touch iOS or Android folders. You also have Expo Go, where you can test your app on a physical device just by scanning a QR Code. Another is the EAS or Expo Application Services, which handles the cloud suite.
