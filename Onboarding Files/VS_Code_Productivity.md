# VS Code and Keyboard Shortcuts
## Top 10 Keyboard Shortcuts in VS Code
1. Ctrl + Shift + P – Open Command Palette
2. Ctrl + P – Quick open file
3. Ctrl + Shift + N – New window
4. Ctrl + / – Toggle line comment
5. Shift + Alt + Down Arrow – Duplicate line down
6. Alt + Up/Down Arrow – Move line up or down
7. Ctrl + Shift + K – Delete line
8. Ctrl + ` – Toggle terminal
9. Ctrl + B – Toggle sidebar
10. Ctrl + F – Find in file

Shortcuts that are most useful in doing daily work are the Shift + Alt + Down Arrow, Ctrl + `, and Ctrl + F. This can improve my efficiency in coding in VS Code since this is reliable in terms of moving around the application, and it is quickly accessible. 

Keyboard shortcuts improve productivity by reducing the need to use the mouse, and they also allow faster execution of commands. Another is that it helps save time, improve your workflow efficiency, and make your coding more comfortable.
