# Understanding React Native Components vs. React Components

The key difference between div and View is that one is used in React Web and the other in React Native. View doesn't render HTML; it renders native UI components on Android and iOS.  As for div, it renders DOM and becomes an actual HTML element in the browser. 
 
StyleSheet.create() improves performance due to its capability to have less code and cleaner code. It is even more scalable, and it is recommended for the production of bigger applications such as Focus Bear. As for the inline style, it is quick, and it is good for small application builds rather than big projects.

On the web, its architecture relies on the CSS engine, the DOM, class selectors, and Cascading Rules. However, in React Native, its architecture has no DOM, no CSS parser, no stylesheet files, and no cascading. The styling of React Native is purely JavaScript and applied directly to native UI components.
